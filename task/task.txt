Ваша задача - создать приложение для заметок на JS в виде веб-приложения. 
Пользователи могут добавлять, редактировать и удалять заметки. 
Список заметок отображается в виде таблицы (HTML-представление может быть различным: таблица, divs и т.д.). 
Столбцами являются время создания, содержание заметки, категория заметки. 
Категории предопределены: "Задача", "Случайная мысль", "Идея".
Заметки в таблице должны также отображать список дат, упомянутых в этой заметке, в виде отдельного столбца. 
Например, для заметки "У меня будет прием у стоматолога 3/5/2021, я перенес его с 5/5/2021" столбец дат будет иметь вид "3/5/2021, 5/5/2021".
Пользователи могут архивировать заметки. Архивированные заметки не отображаются в списке заметок. 
Пользователи могут просматривать заархивированные заметки и разархивировать их.
Также должна существовать сводная таблица, в которой подсчитываются заметки по категориям: отдельно для активных и архивированных. 
Таблица обновляется каждый раз, когда пользователи выполняют какие-либо действия с заметками. 
Сводная таблица отображается на той же странице, что и таблица заметок.

Реализовывать хранение данных не нужно. Достаточно создать JS-переменную, хранящую данные, и предварительно заполнить ее 7 заметками, чтобы они отображались при перезагрузке страницы.

Стремитесь писать чистый код. 
Пишите небольшие функции, чистые функции.
Придерживайтесь принципа единой ответственности. Разделяйте логику визуализации и обработки данных, в идеале - в отдельных файлах.
Давать переменным и функциям описательные имена.


Цель задания - дать возможность лучше познакомиться с языком JavaScript и DOM API браузера. Если вы не знаете некоторых API, необходимых для решения задачи, то можете использовать эти ресурсы в качестве справочных:
